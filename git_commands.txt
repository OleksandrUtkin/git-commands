cd d:/  - перейти в папку
git init - створити в поточному каталозі підкаталог .git який містить всі необхідні файли репозиотрія
git status - подивитись статус репозиторія
git add . - додати всі файли
git commit -m "comment" - закомітити з коментарем
git clone - клонувати репозитрій
git push -u origin master - запушити репозиторій
